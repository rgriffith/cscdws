== Cascade Server Web Services + Rails

A simple Rails application that performs a few simple Cascade Web Services tests.


== Built with...
* Pow[http://pow.cx/]
* Powder gem[https://github.com/rodreegez/powder]
* Bundler gem[http://gembundler.com/] for managing gem dependencies
* Saveon gem[http://savonrb.com] for managing SOAP connections
* Twitter Bootstrap[http://twitter.github.com/bootstrap] framework to make things pretty
* jQuery
