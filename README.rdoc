== Cascade Server Web Services + Rails

A simple Rails application that performs a few Cascade Web Services tests.


== Built with...
* Pow[http://pow.cx/]
* {Powder gem}[https://github.com/rodreegez/powder]
* {Bundler gem}[http://gembundler.com/] for managing gem dependencies
* {Savon gem}[http://savonrb.com] for managing SOAP connections
* {Twitter Bootstrap framework}[http://twitter.github.com/bootstrap] to make things pretty
* jQuery[http://jquery.com]
